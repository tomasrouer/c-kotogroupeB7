Ce répertoire contient l'implementation des codes de notre projet C++.

Nous avons décider de représenter les graphes par les listes d'adjacences, étant la structure de données la plus adaptée.
Nous avons décider de ne pas utilier la librairie STL (standard template library) pour nous forcer à manipuler les pointeurs. De plus cela rend notre code plus transparent.
Par défaut, tous les attributs et méthodes sont mis en "private".

Pour compiler, il suffit d'ouvrir le dossier dans un terminal et lancer la commande make.
Pour executer, il suffit de lancer l'exécutable avec la commande ./test
